Smart-Brain

React Smart Brain Web Application

  • React
  • CSS
  • PostgreSQL
  • NodeJS
  • ExpressJS
  • Postman
  • Clarifai Api
  • Deployment

SmartBrain is a facial recognition web application that allows users to register, log in, and detect faces in images provided via URL. The application incorporates modern web development technologies to provide a seamless and intuitive user experience.

Project Overview

Developed a Face Recognition app utilizing Clarifai API for precise face detection with bounding boxes. Demonstrated strong skills in API integration, image processing, and applying advanced technologies for real-world solutions, showcasing a commitment to staying current in the field of computer vision. The web application is deployed on Netlify and the code is hosted on GitHub.

The creation of SmartBrain was driven by a blend of educational goals, personal interests, and the pursuit of practical skills. SmartBrain served as a playground for honing my React skills. The project's architecture heavily relied on React, showcasing dynamic user interfaces and seamless interactions. Utilizing Node.js for server-side development and PostgreSQL for database management, I delved into the full-stack realm. The integration of these technologies enabled secure user authentication and data storage. The incorporation of the Clarifai API added a layer of complexity and real-world application to the project. SmartBrain's ability to detect faces in images showcased my proficiency in integrating external APIs. Adopting Postman as a testing tool allowed for robust API call testing.

Integrating and understanding the Clarifai API for facial recognition presented complexities, especially given the intricacies of image processing and interpreting API responses. Diligent documentation review, testing with tools like Postman, and watching tutorials have been strategies to overcome this challenge.

In my quest to deepen my understanding of web development and provide a valuable solution for online merchants, I embarked on a journey to build an integrated CMS within an admin dashboard. During deployment of the application on Vercel, encountered a bit of a roadblock when it came time to integrating Prisma using Planetscale. The solution was that I changed the password on Planetscale which automatically made me setup a new database url and I replaced it in .env file of the project. Another issue I encountered was that I installed a bad package so for that I had to delete node modules and pack-locke.json file. After that I did npm install which made the typescript error go away.
